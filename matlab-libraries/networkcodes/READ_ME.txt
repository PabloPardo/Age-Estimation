Install the minmaxfilter function by typing "minmaxfilter_install" at the matlab command window from the directory MinMaxFilterFunctions.

The wavelength and scale parameters used for the gabor filters are adapted from:
Thomas Serre, Lior Wolf, Stanley Bileschi, Maximilian Riesenhuber, and Tomaso Poggio, “Robust object recognition with cortex-like mechanisms,” IEEE Transactions on Pattern Analysis and Machine Intelligence, vol. 29, pp. 411–426, 2007.
